# Custom LLM Web Application

This project is a web-based application that allows users to interact with a customizable AI assistant. Users can chat with the assistant, upload and manage documents, and configure custom instructions to tailor the assistant's behavior.

## Features
- Modern, responsive chat interface
- Persistent custom instructions
- Document upload and management
- Settings panel for configuration
- User-friendly design

## Getting Started
1. Clone the repository
2. Install dependencies with `npm install`
3. Set up your environment variables (see `.env.example`)
4. Run the development server with `npm run dev`
